BEM (Block Element Modifier) CSS Example

This is the introductory paragraph.

This is the main content paragraph.

This is the footer paragraph.

Explanation of class naming:

  1. .paragraph - is the Block representing the common style for all paragraphs.
  2. .paragraph--intro - is an Element class indicating that this paragraph has an introductory style.
  3. .paragraph--main - is an Element class indicating that this paragraph has a main content style.
  4. .paragraph--footer - is an Element class indicating that this paragraph has a footer style.
